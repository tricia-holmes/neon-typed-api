## Getting Started

### Prerequisites
* Before the setup steps, make sure to have [Docker downloaded](https://docs.docker.com/get-docker/).

### Setup
1. Clone the repo
   ```sh
   git clone git@github.com:tricia-holmes/neon-typed-api.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the server
   ```sh
   npm run dev
   ```
4. Run Postgres through Docker
   ```sh
   docker compose up
   ```
